## Marco的个人博客

### 使用 React + Redux + React Router + webpack 搭建个人博客

### 技术栈
#### react + redux + react-router + ES6/7 + webpack + express

### 演示地址
[Marco的个人博客](http://www.hanyuehui.site)


## Usage

### 安装
```
git clone https://github.com/Marco2333/react-projects.git

cd react-projects/blog

npm install
```

#### 开发模式：
```
npm run dev
```

#### 产品模式：
```
npm run build
```
